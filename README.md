# User Management System

A backend application built using Flask to manage users with secure and scalable REST APIs.

## Features
- Create, Read, Update, Delete (CRUD) user operations
- RESTful API architecture
- MySQL database integration
- Input validation and error handling

## Tools, Methods & Frameworks
- Python
- Flask
- REST APIs
- MySQL
- Git & GitHub

## What I Solved
Designed and implemented a backend system to efficiently manage user data using RESTful services.

## Result & Impact
Improved data handling efficiency and provided a scalable API foundation for user-based applications.

## API Endpoints (Sample)
- POST /users – Create user  
- GET /users – Fetch users  
- PUT /users/{id} – Update user  
- DELETE /users/{id} – Delete user  

## How to Run
1. Clone the repository  
2. Install dependencies  
3. Configure MySQL  
4. Run Flask server
